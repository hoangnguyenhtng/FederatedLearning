Privacy by Design

Dữ liệu cá nhân không bao giờ rời khỏi thiết bị. Chỉ model parameters (trọng số) được gửi về server — không thể truy ngược ra dữ liệu gốc.

FedPer + Differential Privacy
🎯

Cá Nhân Hóa

Kiến trúc FedPer giữ lại personal layers riêng cho mỗi user, cho phép mô hình thích ứng với sở thích cá nhân mà vẫn học từ cộng đồng.

Shared Base + Personal Head

🧠 Multi-Modal Adaptive Fusion

Mỗi user học trọng số riêng cho từng modality — tự động ưu tiên nguồn thông tin phù hợp nhất.

📝

Text Embeddings

SentenceTransformer (384-dim)
Reviews + Descriptions
🖼️

Image Features

Semantic Proxy (2048-dim)
Product visual features
🛒

Behavior Features

User interactions (32-dim)
Ratings, purchases, views
⬇️ Adaptive Fusion (Softmax)

🎯 Unified Embedding (384-dim)

💡 Tại sao Adaptive Fusion?

  • User A (text_heavy): Đọc kỹ review → text weight cao
  • User B (image_heavy): Xem ảnh trước → image weight cao
  • User C (behavior_heavy): Mua theo trend → behavior weight cao

w = Softmax(W · [h_text; h_image; h_behavior])

output = w₁·text + w₂·image + w₃·behavior
